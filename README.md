# SGBD 2026.1 - Sistema de Gerenciamento de Banco de Dados

Repositório dedicado à disciplina de **Sistema de Gerenciamento de Banco de Dados** (SGBD) da **Universidade Federal do Ceará**, ministrada em 2026.1.

## Propósito

Este repositório centraliza os trabalhos práticos desenvolvidos como parte da disciplina, com o objetivo de explorar conceitos fundamentais de sistemas de gerenciamento de banco de dados através de implementações práticas e simulações de estruturas de dados essenciais.
